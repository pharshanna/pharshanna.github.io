Full-Stack Android Application

SOUR Suggestions — Predictive Writing Assistant

Data Structures & NLP

A predictive text engine that analyzes Olivia Rodrigo's SOUR album lyrics using a Trie (prefix tree). Type a prefix and the system predicts the most likely next character and next word based on frequency analysis, displaying results with percentage likelihoods. Includes an interactive "Guess the Lyric" game. All operations run in O(L) time where L is the word length — faster than scanning the full dataset every time.

Trie Data Structure Predictive Text Java Frequency Analysis Big O Optimization File I/O

Cookie Cutter — Idle Clicker Game

Android Game Development

A Takis-themed idle clicker game for Android. Tap the screen to earn points with satisfying crunch sound effects and floating "+1" animations. Spend points on upgrades that increase your click multiplier, and unlock passive income that earns points automatically every second. Uses multi-threading with AtomicInteger for thread-safe score tracking across the game loop and UI thread.

Multi-Threading Game Design Java Android Studio Animations MediaPlayer AtomicInteger

Binary Search Tree

Data Structures & Algorithms

A full generic Binary Search Tree implementation in Java supporting insertion, deletion (with in-order successor replacement), search, and three traversal algorithms (in-order, pre-order, post-order). Includes left and right tree rotations — the foundation of self-balancing trees like AVL and Red-Black trees. Built with Java Generics so it works with any comparable data type.

Generics Recursion Java Tree Rotations Traversal Algorithms
